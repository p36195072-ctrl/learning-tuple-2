# learning-tuple-2
Learning Tuple

A beginner-friendly Python project that demonstrates how to use loops to print items from a collection.

📌 Description

This program stores game names inside a list and prints each game using a for loop.

🧠 Concepts Used

Python Lists
for Loop
Iteration
print() Function

💻 Code

games = ["minecraft", "gta5", "fortnite"]

for game in games:
    print(game)
    
▶️ Output
minecraft
gta5
fortnite
